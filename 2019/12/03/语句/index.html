分支结构语句

  1. if…else语句
  2. if 语句
    格式：
    if(布尔表达式)
语句    1        //当布尔表达式结果为ture时执行
语句2        //当布尔表达式结果为false时直接执行这句话
    if只能控制一条语句的执行与否，不过可以使用大括号将多条语句变为块
  3. else 语句
    可以说是if语句的拓展
    格式：
    if(布尔表达式)
语句    1        //当布尔表达式结果为ture时执行
else
语句2        //当布尔表达式结果为false时执行
语句3        //上面某一句语句执行结束后执行
    if语句练习题
  1. switch语句
    格式：
    switch（表达式）{
case 常量表达式1:[语句序列1：]//常量表达式不允许有变量
case 常量表达式2:[语句序列2：]语句序列可以是多条语句，也可以是空语句
default:[语句序列n+1;]//可以不进行default
    }
    一般来说，每个case后都需要break语句来进行结束，否则会进行下坠
    如： 若x=1，则会进行下坠，得出y=2
    而x=3，则不会下坠，得y=3
    当x不为1，2，3时，则会进行default，得出y=4；

终止语句

  1. break
    只能出现在switch和循环体之中
    作用是无条件结束循环结构并且进行下一条语句
  2. continue
    只能出现在循环体中
    作用是无条件结束当前循环并且进行下一次循环
