表达式练习题

2019-12-03

练习题

  1. 以下程序段运行结果为（ D）
    double a = 5.5, b = 2.5;
System.out.println((int)a + b / b);
    A）6.5 B）6 C）5.5 D）6.0
  2. 假设在程序中a、b、c均被定义成整型,并且已赋大于1的值,则下列能正确表示代数式 的表达式是（ B）
    A）1/abc B）1/(abc) C）1.0/a/b/c D）1/a/b/(float)c

  1. 设有int x=11；则表达式(x++*1/3)的值是（ A）
    A）3 B）4 C）11 D）12

  2. 若有以下程序段,执行后c3中的值是（ A）
    int c1=1,c2=2,c3； c3=1/c2*c1；
    A）0 B）0.5 C）1 D）2

  3. 若有定义:int a=8,b=5;double c；执行语句c=a/b+0.4；后，c的值为（ B ）
    A）1 B）1.4 C）2.0 D）2

  4. 若有定义double x=1,y; ,则表达式y=x+3/2的值是（ C ）
    A）1 B）2 C）2.0 D）2.5

  5. 设变量x为float型且已赋值,则以下语句中能将x中的数值保留到小数点后两位,并将第三位四舍五入的是（ C）
    A）x=x100+0.5/100.0； B）x=(x100+0.5)/100.0；
    C）x=(int)(x100+0.5)/100.0； D）x=(x/100+0.5)100.0；

  6. 若变量a已说明为float类型,i为int类型,则不能实现将a中的数值保留小数点后两位,第三位进行四舍五入运算的语句是（ D ）
    A）a=(a100+0.5)/100.0;a=(int)(a100)/100.0； B）i=a100+0.5;a=i/100.0；
    C）a=(int)(a    100+0.5)/100.0； D）a=(a/100+0.5)*100.0；

  1. 以下程序段运行结果为（ B ）
    int a = 5, b = 6, c = 7, d = 8;
boolean m = true, n = true, k;
k = (m = a > b) && (n = c > d);
System.out.println(m + "," + n);
    A）true,true B）false,true C）true,false D）false,false
  1. 以下程序段运行结果为（ C ）
    int a = 1, b = 2;
boolean m = false, n = false, k;
k = (n = b > a) || (m = a < b);
System.out.println(m + "," + n);
    A）false,false B）true,false C）false,true D）true,true
  1. 以下程序段运行结果为（ C ）
    int a=-1,b=4;
boolean k;
k=(a++<=0)&&(!(b--<=0));
System.out.println(k + " " + a + " " + b);
    A）false 0 3 B）false 1 2 C）true 0 3 D）true 1 2
  1. 以下程序段运行结果为（ D ）
    int a = -1, b = 4;
boolean k = (++a < 0) && !(b-- <= 0);
System.out.println(k + " " + a + " " + b);
    A）true 0 4 B）true 0 3 C）false 0 3 D）false 0 4
  1. 以下程序段运行结果为（ D ）
    int a = 1, b = 2, c = 3, d = 4;
System.out.println(a < b ? a : c < d ? a : d);
    A）4 B）3 C）2 D）1

  1. 以下程序段运行结果为（ C ）

    int a = 5, b = 4, c = 6;
System.out.println(a > b ? ( a > c ? a : c ) : b);

    A）5 B）4 C）6 D）不确定
    c=6

  2. 以下程序段运行结果为（ D ）

    int w, x, y, z, m;
w = 1; x = 2; y = 3; z = 4; 
m = (w < x) ? w : x; 
m = (m < y) ? m : y; 
m = (m < z) ? m : z;
System.out.println(m);

    A）4 B）3 C）2 D）1

  1. 以下程序段运行结果为（ A ）
    int a = 2, b = 5;
System.out.println(a > b ? a++ : b++);
    A）5 B）2 C）6 D）3
  1. 以下程序段运行结果为（ D ）
    int i, j, k, a = 3, b = 2;
i = (--a == b++) ? --a : ++b;
j = a++;
k = b;
System.out.println(i + "," + j + "," + k);
    A）2,1,3 B）1,1,2 C）4,2,4 D）1,1,3
