4、求1-1/3+1/5-1/7+…+1/2n-1

5、求1+1/(1+2)+…+1/(1+2+3+…+n)

6、求n!

7、求1+2!+3!+…+n!

8、求x的n次方

9、求下面式子

           x2      x3    xn（2，3，n是次方）
求 1 + x + ── + ── + … + ──
           2!     3!     n!

10、求下面式子

         3x2    5x3    7x4         (2n-1)xn

            2!     3!      4!         n!
